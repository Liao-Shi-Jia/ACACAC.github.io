<br> 1. 得到 __常量__：
<br>① 格里高利历 ___400___ 年的天数为 ___146097___
<br>② ___4713.1.1___ ~ ___1582.10.4___ 的天数为 ___2299160___ 
<br>方法如下
 ```c++
int n, ans = 0;
scanf("%d", &n);
ans += (n / 4) * 366;
ans += (n - (n / 4)) * 365;
  ```
